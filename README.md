# WA Navbar Hider

A lightweight extension for Chromium-based browsers that removes selected navbar items from **WhatsApp Web**.

Removes:

- Status
- Channels
- Communities

---

## Installation

1. Open your browser extensions page:

   - Chrome → `chrome://extensions/`
   - Brave → `brave://extensions/`
   - Edge → `edge://extensions/`
   - Opera → `opera://extensions/`

2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the project folder

Open `https://web.whatsapp.com/` and the navbar items will be removed automatically.
